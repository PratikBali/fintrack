# **App Name**: FinTrack Pro

## Core Features:

- Customer-Supplier Balance Management: Add customers and suppliers with contact details, record credit/debit transactions, track balances, send payment reminders, and generate reports.
- Group Expense Splitting: Create groups, add shared expenses with splitting methods (equal, percentage, custom), track who owes whom, and settle up balances.
- Personal Expense Tracking: Add personal income and expenses, categorize transactions, set budgets, track goals, and view spending analytics with charts.
- Manual Input: Allow users to input their income and expenses manually.
- Receipt Scanning: Enable OCR to scan receipts and automatically extract information using a tool, such as amount, date, and vendor.
- Categorization: Categorize expenses (e.g., food, transport, shopping).
- Spending Analytics: Display spending patterns through charts and graphs.
- Budgeting: Set monthly budgets for different categories.
- Reporting: Generate reports of income and expenses over specified periods.
- Currency Support: Enable multiple currency support with real-time exchange rates.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for trust and security in financial management.
- Background color: Soft, light gray (#F0F4F7), creating a clean and unobtrusive backdrop.
- Accent color: Warm orange (#FFAB40) for calls to action, providing a friendly and inviting feel.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and neutral look.
- Use minimalistic and clear icons to represent different expense categories.
- Maintain a clean and intuitive layout, prioritizing key information and easy navigation.
- Subtle animations on data updates and transitions for a smooth user experience.
